1. Установить программу R-3.0.2-win
2. Открыть скрипт  clearCode.R
3. В коде на ~5 строке указать путь к папке, где находятся все рабочие файлы. (PS. альтернативный вариант: удалить эту строчку и переместить все файлы в "Мои Документы")
4. Запустить скрипт на исполнение (Ctrl +A ,затем Ctrl+R)
5. Для запуска функции написать:
	fsalary()- если необходима информация по всем продукциям и квалификациям, 
	fsalary(Product=N) - если необходима информация по продукту, где N - id продукта
	fsalary(Qualification=N) - если необходима информация по квалификация, где N - id квалификации
